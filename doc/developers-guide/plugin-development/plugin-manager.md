`reckless` is a plugin manager for Core Lightning that you can use to install and uninstall (and more!) [plugins](doc:plugins) with a single command.

Please refer to the `reckless` README for more information: [README.md](https://github.com/ElementsProject/lightning/blob/master/plugins/reckless-plugin/README.md)
